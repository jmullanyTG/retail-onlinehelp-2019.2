Substituting size curves

To substitute one site’s curve for another:

  1. Select the size curve header you want to make substitutions in from the list in the top area of the Size Curves Screen.

  2. Ensure the Override Allowed? field is checked.

  3. In the lower area of the Size Curves screen, select the desired site from the drop-down menu in the Override Site field of the site whose curve you want to override then click Save.

    Note: sites that have an Override Site assigned and Saved will not appear for selection in the Override Site drop-down menu

    Note:to clear the Override Site prior to Saving, simply select the drop-down entry with a blank Site ID and No Substitution in the Description

 

The Contribution values of the overriden site’s size curve will now match those of the Override Site. The Clear Override field will be conditionally formatted orange, indicating an override has been made.

Note: Is Eligible? settings for the Override Site will also be copied
