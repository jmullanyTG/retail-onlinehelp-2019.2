Site Contribution grid

The site contribution grid at the bottom of the Product Model tab displays the percent contribution to total from each of the sites.

  • Site Name – The name of the site.
  • Site Code – Unique identifier of the site.
  • Hierarchy – Site specific hierarchy information.
  • Contribution – Percentage contribution for each site.
  • Raw Contribution Sort – A helper field used for data sorting. Values are irrelevant to contribution calculations.

Depending on the highlighted product model, users will be able to view the contribution per site to the product model. Users may pivot this data and if desired, view contribution as a graph.
