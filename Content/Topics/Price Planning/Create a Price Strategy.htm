Create a Price Strategy

Within the Price Manager Configuration Strategy screen, click New to create a new Price Strategy, enter the required fields, and click Save.

Restricting access to a new Price Strategy

By default, all Users and Roles have access to a new Price Strategy. Removing access to a new Price Strategy does not affect any current Pricing Actions.

NOTE: The list is filtered to display only active Users and Roles that have the User Right Add Price Action.

To restrict access to a new Price Strategy:

  1. Click on the User Access field to display the Remove User Rights dialog.

  2. Select the Users and Roles that will be restricted from having access to the new Price Strategy.
  3. Click Save.

Field descriptions

Field

Description

Price Strategy Name (Required)

Enter a descriptive label to summarize the purpose of the Price Strategy.

Price Strategy Description (Optional)

Enter a summary of the Price Strategy.

Active (Optional)

Unmarking the checkbox will remove the ability to configure the Calculation in any new Price Strategies.

Is Default (Optional)

Ability to designate a Price Strategy to be set when creating a new Pricing Action.
User Access Clicking on this field opens the Remove User Rights dialog in which you can grant or restrict User and Role access to Pricing Strategies.
