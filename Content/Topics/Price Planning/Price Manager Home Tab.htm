Home Tab

The Home tab displays your pricing plans and their related price actions. It also contains a list of product changes that have been imported into the system that require your attention.

 

Product Changes grid

 

This grid displays product changes that could not be automatically applied to price actions. For each product change, you have the option to take the following actions:

Action

Description
Add to existing price actions Adds selected product changes to existing price actions.
Add to a new pricing plan Creates a new pricing plan and adds the selected product changes to the new plan.
Dismiss the product change Removes the product change from the grid.

 

When the system imports changes to products (including inventory, item master, variant, cost, and price changes), Price Manager attempts to update the price actions associated with the changed products. If Price Manager is unable to automatically update a price action with product changes (either because the changed products are not associated with any of your existing price actions or because Price Manager is not configured to automatically update import changes).

Actions

You can perform the following actions from the Home tab.

Action

Description
Open a pricing plan Double-click on a plan in the Plans table to open the plan in a new tab. (You can also select the plan and click Edit on the Plan toolbar.)
Create a new pricing plan Click New on the Plan toolbar to create a new pricing plan.
Delete a pricing plan Click Delete on the Plan toolbar to delete a selected pricing plan.
Open a price action Double-click on a price action in the Price Actions table to open the action in new tab. The action will be opened in the context of its parent plan.
