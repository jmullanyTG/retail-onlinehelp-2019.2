Adding metadata in Site Manager

You can add metadata as columns in Site Manager documents and use it in site filters to find sites which match specific criteria.

Click the Choose button on the toolbar to display the Choose Fields selection window. Within this window, select the metadata you want to include.

The newly included metadata displays in the grid in the top half of the screen.

Exception-based filtering

You can use metadata to create exception filters that identify sites in need of additional configuration, such as:

  • Sites that have not yet been assigned to a site cluster
  • Sites that do not appear in any Allocation location strategies.
  • Sites that have not been added to a supply chain.
