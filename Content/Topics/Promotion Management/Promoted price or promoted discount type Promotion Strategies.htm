Promoted price or promoted discount type Promotion Strategies

For Promoted Price and Promoted Discount type Promotion Strategies there are three ways to add products. The three methods of adding products are not mutually exclusive and all three methods can be used within a Promotion.

Adding a Variant

  1. Click Add Variant.

  2. In the Add Variants dialog, enter a product code (for Variant or Item Master, depending on the global policy) in the Products field or select the Add Variants From Clipboard button.

    NOTE: Variant codes can be copied using the right-click menu option Copy Column Cell(s).

  3. Click Ok.

Using a filter to add products

  1. Click Select to Add.

  2. In the Variants dialog, click the dropdown arrow in the Filter field and select a filter.

    NOTE: If an appropriate filter does not exist click, click on the black hammer icon to create a new filter.

  3. If required, select the check box Select only some items from the current filter for individual product selection from the selected filter.
  4. Once the appropriate Filter is selected, click Ok.

Adding a Variant group

  1. Click Add Variant Group.
  2. If required, click Filter to change which filter is being used.
  3. In the Select Variant Groups to Add dialog, select the check box for each group to be added to the Promotion.
  4. Click Ok.

NOTES:

  • If the group you wish to add is not available, first ensure the filter selected in the pop up is for “All Variants/Item Masters”. If still unavailable, navigate to Content Manager to create the group you require for the Promotion.
  • These groups are dynamic, meaning the system will update the associated products included in the group dependent on the filter criteria used to create the group. This update will only occur as long as the Promotion is in an unlocked state.
  • When adding a group to a Promotion, the system creates a copy of the original group, including the filter criteria used to define the Group, so that changes can be made at the Promotion level without affecting the originating product group.

Additional Information

For information on creating, editing, and deleting filters, refer to the Filters section in the online help.
