Promotion Validation Statuses

The Validation Status of a Promotion is indicated using icons.

Validation Severity icons

The following are the Validation Severity icons for failures not yet reviewed:

Validation Severity icon Description
Information Provided
Warning Encountered
Error Found

The following are the Validation Severity icons for failures that have been reviewed:

Validation Severity Icon Description
Information Reviewed
Warning Reviewed
Error Reviewed

Promotion Validation Log

Double-click a Validation Severity icon to display the Promotion Validation Log.

The Promotion Validation Log does not limit the number of records to be returned.

The icons are as outlined below (most severe to least):

Icon Description
An error was found so you must review the Promotion to resolve the problem.
A Validation warning was encountered.
Validation ran successfully but additional information is available to review.
A Calculation is required before a Validation Status can be assigned, so you must either manually open the Promotion and recalculate, or it can be recalculated by an overnight process.
The Promotion is successfully validated.

Validation is done at the Site Cluster level so it is possible for one Promotion to have one Site Cluster with a warning, another with information to review, etc. The Validation Status displayed at the Promotion level will be a roll up of the most severe Validation Statuses at the Site Cluster levels.
