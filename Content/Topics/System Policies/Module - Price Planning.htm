Advanced Validation Log PriceManager.AdvancedValidationLog When this policy is set to 'False', the system will display failed Validations at the Inventory level. When the policy is set to 'True', the system will group Inventory failed Validations up to the Variant level and Site (i.e. suppressing Item Master). The system will also allow overrides to be acted upon at the Variant level if configured.

Auto update Price Action on import changes

 PriceManager.AutoUpdatePriceActionsOnImportChanges

Controls the system to implement automatic updates for imported product changes. By default, the system is set to ‘False’ requiring you to review changes before updating the Price Action with the new information from the user interface.

If set to ‘True’ the system will automatically update Price Actions with imported changes that affect Pricing Actions with the same Product and Site combination.

Config View Collection Set

PriceManager.ViewCollectionSetForPriceManagerConfig

This policy determines the user interface to be used for the Price Manager’s Configuration.

Data Initialize Batch Size

  

This policy determines the number of records to return, at one time for performance reasons, when initializing a Price Action. The system default is 1000 Item Masters records. This will need to be modified based on the available server configuration.

Price Manager Inventory Meta Data View

 PriceManager.DataInitializeBatchSize

Inventory Meta Data View PriceManager.InventoryMetaDataViewID Set ViewID for Price Manager inventory view meta data

Plan Start Date Number Of Days Out

 PriceManager.PlanStartDateNumberOfDaysOut

Set default number of days out from current for start date field in Price Manager. This can only be overridden (i.e. backdated) by users with the ‘Emergency Pricing’ user right, which will allow a date to be selected as early as tomorrow.

Price Action View Collection Set

 PriceManager.ViewCollectionSetForPriceAction

A View Collection Set that may be configured to display information in context of a Price Action (e.g. reports).
Price Planning View Collection Set PriceManager.ViewCollectionSet A View Collection Set that may be configured to display product information pertinent to Price Planning.
Pricing Plan View Collection Set PriceManager.ViewCollectionSetForPricingPlan

Sets whether or not to display a pricing plan view collection set as a tab in the Price Manager's pricing plan screen.

Value = 0 does not display a view collection set tab.

Value = n (where "n" = a view collection set ID) displays the corresponding view collection set as a tab in a pricing plan screen. The tab name is the name of the selected view collection set.
