Importing Configuration Migration XML export files

As a system administrator, you have the ability to script updates to multiple systems removing the human element (error) and improving efficiency in upgrade timing. The File Import Utility includes parameters to import Configuration Migration XML export files.

Parameters required to import Configuration Migration XML export files

The following parameters are required to import Configuration Migration XML export files:

  • path
  • COMPANYNAME
  • BLLURL
  • USERNAME
  • PASSWORD
  • DIR
  • INCLUDESUB
  • TYPE

Example

$cmd = "C:\JustEnough\3.7.0\Utilities\Import\JustEnough.Utility.Import.exe COMPANYNAME: MyCompany BLLURL: http://localhost/justenough.bll/ USERNAME:IOAdmin PASSWORD:MyPassword DIR:C:\JustEnough\3.7.0\ConfigExport.XML INCLUDESUB:N TYPE: ConfigMigration
