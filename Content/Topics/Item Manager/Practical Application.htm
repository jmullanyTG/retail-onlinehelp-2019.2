Practical Application

Opening and building an Inventory Manager view is a relatively simple task which gives users a high degree of flexibility. The easy to configure views can be used to create custom reports for any information within Demand Management.

Views may be created and opened in separate tabs within the Inventory Manager module. Upon opening a new view, users will be prompted to select a filter of inventories.

This can be a static filter, or one of the many dynamic filters, all of which are managed through the filters module.

Fields are organized by category and may be included or excluded by category, or individually.

The search box near the top of the choose fields window can be used to filter the fields and find specific measures.
