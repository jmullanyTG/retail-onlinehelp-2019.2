Allow Editing of Promotion

Provides access to edit the Promotion Price Cluster Sites, Price Cluster Promotion Strategies, Clone the Promotion, modify Price Details, create and modify Bids, add and modify Companion Items and view the Analysis screen.

Note: Without this User Right, you may still view Promotion information in the Price Cluster Items, Price Details, Bids, Companion Items and Analysis screens by double clicking on the Promotion Name from the Promotion List screen

NOTE: Some fields may appear to be editable but are not, if this User Right has not been granted
