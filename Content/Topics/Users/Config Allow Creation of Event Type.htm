Config - Allow Creation of Event Type

Provides the ability to specify an Event Type of default attributes (e.g. Geography Set, Slot Set Templates, Cost Types and their default amounts, labels, etc.) that can be grouped and applied to new Events created using the Event Type. Some of these default attributes can then be customized at the Event level.

NOTE: This User Right should only be granted to trusted Super Users
