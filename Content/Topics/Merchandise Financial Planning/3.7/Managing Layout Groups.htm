Managing Layout Groups

Layouts can be organized into groups, which makes it easier to find the layout you are looking for when restoring a layout or data layout. You can add, delete, and edit groups from the Save Layout dialog box. Changes are reflected in both the Restore Layout and Restore Data Layout context menus.

Note: Administrator rights are required to manage groups.

  1. Right-click in the data area of the grid and choose Save Layout.

  2. In the Save Layout dialog box, click the "..." button next to the Group list.

  3. In the Group Management dialog box:

    • Click New to create a new group.
    • Click Edit to edit the selected group.

    • Click Delete to delete the selected group. Any layouts contained in a deleted group are moved to the main (ungrouped) list of layouts.

  4. Click OK in the Group Management dialog box to save your changes.
  5. If you want to make changes to the current layout, such as saving it to a new group, click OK in the Save Layout dialog box. Otherwise, you can click Cancel without affecting the changes made to your groups.
