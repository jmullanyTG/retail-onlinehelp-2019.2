The Financial Plan Maintenance Window

The Financial Plan Maintenance window lists the existing plans you have access to. From this window, you can open plan documents as well as perform administrative actions such as creating, branching, merging, and initializing plans.

Getting here

  • Click Plan > Merchandise Financial Planning > Manage on the toolbar.

Opening a plan document

  1. On the Financial Planning toolbar, click Manage.
  2. In the Financial Plan Maintenance window, select the plan you want to open from the plan list.
  3. Click Ok at the bottom of the window to open a document for the selected plan.

Actions

The following actions are available:

Action Description
New Creates a new plan.
Edit Edits the settings of the plan selected in the plan list.
Initialize Seeds/populates data values in the selected plan according to the measure set selected for the plan. This action is only available to plans with the status, "Created".
  • Regardless of the selected seed type, metrics that would be seeded with negative values are seeded with zeroes instead.
  • Once a plan has been initialized, its status changes to "Initialized".
Re-Initialize Re-initializes the selected plan. A plan should be re-initialized when its master data changes in the data feed (a re-class, for example).
Close Deletes the selected plan.
Re-Seed Re-seeds future periods within the selected plan.
  • A master plan is re-seeded from calculation as defined by it seed type.
  • A branch plan is reseeded from its "parent" plan.
  • Regardless of the selected seed type, metrics that would be seeded with negative values are seeded with zeroes instead.
Branch Creates a branch of the selected master or branch plan.
Merge Merges the selected branch plan into another branch or master plan.
Submit Submits the selected branch plan for approval.
Approve Approves the selected plan.
Reject Rejects the selected plan.
Rebalance

Takes the desired values from the plan version to be continued with and homogenizes other versions to them. For more information, see

Global Groups Manages the site and product hierarchies used in your financial plans.
User Access Controls user/role access to the selected plan, including product, channel, and measure sets within the plan.
Merge Measure Sets Controls which set of metrics are merged from the selected plan.
Refresh Updates the information displayed in the Financial Plan Maintenance window.
