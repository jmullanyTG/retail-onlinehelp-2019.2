Event Manager

The following enhancements are now available in the Event Manager solution.

Events with a Promotion marked as PrivateDMDEV-59751, DMDEV-59748, DMDEV-59747, DMDEV-59745

Promotions can now be marked as Private, Shared, or Public. This gives Promotion owners the ability to hide and protect sensitive content by marking particular promotions as Private or Shared (with only particular users). A logged in user will be able to view Promotions, in the Event solution, that are marked Public or are shared with the logged in user.

A private or unshared Promotion will not be seen in the Available Promotion panes and is masked if used in an Event to hide its specific details as well as to make clear that a Slot is assigned to a Promotion. This masking applies to the Event List, Layout and Version screens but will not affect the Version Report.

Previewing private Promotions owned by another userDMDEV-61883DMDEV-61761

Whether in the Event List or Layout view, previewing a private Promotion owned by another user is restricted to only those users with whom the Promotion is shared. The Promotion preview pane masks the details for a private Promotion that has not been shared with a user who double-clicks on the private Promotion.

Restrictions on modifying private Promotions owned by another user

For promotions owned by another user that are private to the logged in user, the logged in user cannot:

  • delete Event Slot Sets which have at least one Promotion that has not been shared with the logged in user. DMDEV-62080
  • modify the Slot Set Template as it may cause the private Promotion to be removed from the Slot Set (i.e. page)

  • remove or unassign private Promotions

    NOTE: For Promotions that are private to the logged in user, the logged in user may still select Remove All on the Promotion menu. However, Remove All will only be applied to Promotions that the logged in user has access to. Promotions that are private to the logged in user are not removed.

  • merge or split Slots with private Promotions or where private Promotions are adjacent and would be affected

    NOTE: In Layout Edit mode, Promotions that are private to the logged in user cannot be split or merged. Promotions that are adjacent to a private Promotion cannot be merged to the private Promotion.

    For a public Promotion adjacent to a private Promotion, the purple line does not appear when the public Promotion is selected and the public Promotion cannot be merged. Also, logged in users cannot split a Promotion that is private to them.

Cloning an Event with Promotions DMDEV-61869

A Promotion’s privacy options can be retained when cloning an Event that has Promotions to maintain privacy for other vehicle types or time frames. In the Clone Event dialog, select the Include Promotions check box to keep the Promotion Privacy options including the shared list of Users and Roles.
