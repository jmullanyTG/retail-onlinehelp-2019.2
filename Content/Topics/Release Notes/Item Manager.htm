The following enhancements are now available in the Inventory Manager solution.

Enabling a status bar in the Forecast Viewer grid DMDEV-61861

A new right-click menu option labeled Status Bar allows you to view SUM, AVG, COUNT, MIN, and MAX values in a status bar at the bottom of the Forecast Viewer grid. The Forecast Viewer grid is found on the Details pane in the Inventory Manager module.
