Cluster Site Manager Screen

In the Cluster Site Manager screen, users can launch the Cluster Set Management window, calculate and view specific cluster sets in both detail and chart modes, make manual adjustments to site cluster assignments, and arrange multiple cluster sets in the view for easy comparison.

Cluster sets the user has chosen to view appear as tabs with the cluster set name just below the toolbar.

Cluster Set screen toolbar buttons and header fields

Toolbar buttons

Manage: launches the Cluster Set Management dialog

Save: saves changes made to the User Cluster and Apply User Value fields since the last Save

Undo: undoes changes made to the User Cluster and Apply User Value fields since the last Save

Calculate: calculates the cluster assignments for all stores included in the cluster set (based on the parameters and criteria specified during cluster set configuration) and populates the Cluster field with these assignments.

Note: You can calculate multiple cluster sets at once from the Cluster Set Management window. From there, just click Allow Multiple and select the desired cluster sets and/or cluster set partitions.

Configuration: launches the cluster set Configuration dialog

Refresh: refreshes the current view

Header fields

Name: The name of the cluster set being viewed

Description: The description of the cluster set being viewed

Cluster Type: The cluster set type. For more information on cluster set types, see Cluster Set Configuration.

Last Computed: displays the date the cluster set was last calculated

Tab header right-click menu options

Right-clicking a document tab displays the following options:

  • Close: closes the cluster set document you right-clicked
  • Close All But This: closes all open cluster set documents except the one you right-clicked; only available when multiple documents are open

  • Float: converts the document tab to a window that can be sized and positioned on the screen (or other screens, in multiple monitor configurations)

    Note: floating can also be achieve by simply clicking and dragging the document header without right-clicking and selecting . After floating, click the document header and drag the document to the desired location. Right click the document header and click Dock as Tabbed Document to return the document to its tabbed position. You can also hover over the directional buttons of the positioning reticle that appears while dragging to see possible docked locations for the floated window.

  • New Vertical Tab Group: configures all documents currently being viewed vertically side by side; after clicking this option, right click the document header again and select Move to Previous Tab Group to return to the last screen configuration

  • New Horizontal Tab Group: configures all documents currently being viewed horizontally stacked; after clicking this option, right click the document header again and select Move to Previous Tab Group to return to the last screen configuration

Detail tab

The Detail tab displays the site codes and descriptions of all stores in the cluster set and their cluster assignments. You can edit cluster assignments.

Tab header fields

Name: The site name and number.

Description: The site description.

Cluster: The system calculated cluster assignment for the site.

User Cluster: Editable dropdown. Displays/sets the user override cluster assignment for the store. Not required.

Apply User Value: Editable checkbox. Sets whether the system uses User Cluster in place of Cluster as the Effective Cluster or not. Default is checked.

Effective Cluster: The effective cluster assignment for the store.

Grid options

Grid options include:

(The following grid options were introduced with the v2019.2.1 Service Pack.)

Select All: Selects and highlights all Site records within the Site Cluster Detail grid.

Deselect All: Deselects and un-highlights all Site records within the Site Cluster Detail grid.

Select from Clipboard: Selects and highlights all matching Site records in the Site Cluster Detail grid.

Deselect from Clipboard: Deselects and un-highlights all matching Site records in the Site Cluster Detail grid.

Chart tab

The Chart tab displays all stores in the cluster set as data points against X and Y axes for all Attributes defined in Cluster Set Configuration.

  • X-Axis Attribute: user editable dropdown; selects an Attribute from those defined in Cluster Set Configuration to plot on the X (horizontal) axis of the graph
  • Y-Axis Attribute: user editable dropdown; selects an Attribute from those defined in Cluster Set Configuration to plot on the Y (vertical) axis of the graph
  • Legend: user editable checkboxes; displays one color-coded checkbox per cluster, corresponding to the colored data points that are assigned to that cluster on the chart; unchecking hides the related data points

  • Data Points: colored dots and stars within the chart area; dots correspond to clusters of the same color in the legend, stars represent the centroids (K-Means types) or medoids (K-Medoids types) for the distance calculations used to assign stores to clusters. Aligning the cross hairs on the chart to a specific data point (done simply by moving the cursor on the chart) shows a pop-up of the store associated with that point as well as the specific values associated with the X and Y axes at the left and bottom of the chart, respectively

  • Show Labels: user editable checkbox; toggles the display of data point labels for all points on the chart; defaults to unchecked
