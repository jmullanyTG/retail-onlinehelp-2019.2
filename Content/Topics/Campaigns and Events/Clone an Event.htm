Clone an Event

To clone an existing Event, display the existing Event by navigating to Influence > Events and double clicking an event within the Events grid on the List screen.

The Event displays as a separate screen.

  1. Click the Clone button on the toolbar to display the Clone Event wizard.

  2. Complete the fields in the wizard as required.

    If you select the Include Promotion check box when cloning an Event with associated Promotions, then the privacy options are carried over from the original Promotions. Also, the last Promotion Owner is retained in the new Promotion being created by the cloned Event.

    NOTE: For more information on the individual fields, see Create an Event (wizard).

  3. Click Ok.
