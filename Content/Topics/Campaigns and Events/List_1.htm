List

The Slot Set List view displays the Slots in the selected Slot Set as rows in a grid. Access to this screen depends on the user’s rights.

For each Slot, the attributes are displayed as columns.

Every Slot Set has a Position 0 Slot. This is where all unadvertised Promotions will be located for the Slot Set. Position 0 does not have an associated Item Display Format, Space, and the Status is ‘Unassigned’.

Private Promotions

If the logged in user selects a Promotion that has not been shared with them, then:

  • the Promotion name in Assigned Promotion displays as Private.
  • All fields except Item Display Format and Space are masked.
  • the ability to modify Item Display Format and Space is disabled.

Displaying the View/Edit Promotion screen

To preview a particular slot, double-click on a row in the grid to display the Promotion preview screen.

Previewing a Promotion

Selecting a slot in the grid refreshes the Slot Preview Pane (located to the right of the Promotion pane that displays the Promotions available to be added to the Event). If the Promotion has been shared with the logged in user (the Promotion’s privacy option is Public or Shared with the logged in user), the Promotion preview dialog will be displayed, but is read-only.

Previewing private Promotions

The Promotion preview pane is masked and displays the following graphic if the logged in user double-clicks a slot with a Promotion that has not been shared with them.

Selecting Geographies and available Promotions

Selecting a Slot in the grid, will refresh the Geography and Available Promotions panes below the grid.

Geography pane

The Geography pane indicates which Geographies will be automatically assigned to the Promotion when it is added to the Slot. If the logged in user selects a Promotion that has not been shared with them (the Promotion is private to other users), then the Assign button is disabled.

Available Promotions pane

The available Promotions pane indicates which Promotions are available to add to the Slot. A Promotion with a check mark indicates that it has been added to the Slot Set but not necessarily to the selected Slot. If a Promotion is assigned to the selected Slot, it will be the only one displayed in the Promotions pane.

The Available Promotions pane displays Promotions with Sales Dates that overlap the Event Sale Dates. There are several buttons available above the Promotions pane:

NOTES: The available Promotions pane:

  • does not display Promotions that have been designated Private.

    Promotions that have not been shared with the logged in user (the Promotion is private to other users) do not display in the Available Promotions pane though they do display as Private Promotions in the Slot Set List view to denote the Slot has been populated.

  • displays Promotions designated as Public.
  • displays Promotions that have been shared with the logged in user or the role the logged in user belongs to.
  • displays Promotions that are Private to the logged in user.

Depending on the Global Policy Promotion Double Click Action setting, double clicking on an assigned Promotion in the grid will load the Promotion in either a window or in the Promotions solution.

Promotions menu

The following functionality is available in the Promotions menu on the toolbar:

   

The following functionality is disabled when a Promotion with a privacy option of Private is selected.

NOTE: Remove All will only be applied to Promotions that the logged in user has access to. Promotion’s that are private to the logged in user are not removed. For example, if all the Promotions are shared except for the Promotion in slot 6, then all the Promotions are removed except for the private Promotion in slot 6.

Promotion: New

If a Slot, has one or more Geographies available and the user has the appropriate Workflow permissions and rights, the ability to create a new Promotion is available. Clicking on New will open the New Promotion Wizard window. Enter the required fields and click Save and Close. For more information, see Create a New Promotion.

Promotion: Remove

If a Promotion is selected in the Slot Set, the Remove button is available. Click on this button to remove the selected Promotion from the Slot. This will only remove the selected Promotion. Any other Promotions will still be associated with the Slot. A different Promotion may be added to the newly open Geographies in the Slot.

Promotion: Remove All

If a Promotion is selected in the Slot Set, the Remove All button is available. Click on this button to remove all Promotions from the Slot. The Slot is considered open after this action.

NOTE: Promotions that are private to the logged in user are not removed and, therefore, the Slot is not open.

Promotion: Unassign

If a Promotion is selected in the Slot Set, the Unassign button is available. Click on this button to move the Promotion from an assigned Slot to the Unassigned Position 0. The Promotion will move to Slot 0 and still be associated with the Slot Set.

View

If a Promotion is selected in the Slot Set, the View button is available. Click on this button to view the Promotion in Promotion Manager. If the user has the appropriate rights, modifications can be made to Promotion and saved. Changes will be reflected in the Event after the Slot Set is refreshed.

Content

For any Promotion in an Event, it is possible to modify the Content and Images associated with it using the Content Editor. For more information, see Content Editor.

Slot Set grid columns

Field

Description

Slot Number (Read Only)

The Position of the Slot in the Slot Set.

Assigned Promotion (Read Only)

If a Promotion has been added to the Slot, the Promotion Name will be displayed.

Content Version (Read Only)

If a Promotion has been added to the Slot and more than one Content Version exists for it, each Content Version will have its own row in the grid. The Content Version Name will be displayed in this column.

Item Display Format (Optional)

The Item Display Format to be used across all Promotions in the Slot. Clicking on this field displays a dropdown list populated with the available Item Display Formats. The available Item Display Formats are filtered to be within corporate compliance to ensure consistent messaging to customers via the Slot Set Template Configuration screen.

NOTE: If an Item Display Format is assigned that is no longer applicable to the Slot Set Template the option will remain until it has been substituted with a different Item Display Format. The invalid Item Display Format will be no longer available once another Item Display Format has been selected and saved.

Space (Optional)

Indicates the Space assignment for the Slot. This may be modified by selecting a new Space from the dropdown list. Changing the Space assignment may affect user permissions to the Slot.

Status (Read Only)

If a Promotion has been added to the Slot, the Status is ‘Assigned’, otherwise it is ‘Open’. It is possible for a Slot to have Statuses of ‘Assigned’ and ‘Opened’ if not all of the available Geographies are assigned a Promotion.

Notes (Optional)

Enter instructions or additional information related to the treatment of the assigned Promotion directly into the grid. This is not editable if the Slot is ‘Open’.

Start Date (Read Only)

Displays the Promotion Start Date.

End Date (Read Only)

Displays the Promotion End Date.

Promotion pane

The following table describes the functionality in the available Promotions pane:

Action

Description

Add to Event

Clicking this button will assign the selected Promotion to the selected open Slot in the current Slot Set. Access to this button may be restricted depending on the user’s permissions and the Slot’s Space assignment.

View

Selecting this button will open the selected Promotion in the Promotions solution.

Filter

Filter can be used to limit which Promotions are displayed in the pane. By default all Promotions with dates that overlap with the Event Sale dates are displayed. To limit Promotions, select from the dropdown list of existing Promotion Filters. If an appropriate Filter does not exist click on the black hammer icon to create a new Filter that will better match the list of Promotions to display in the pane. Once the appropriate Filter is selected, click Ok.

Unassigned

If the Slot Set contains any Promotions that are associated with the Slot Set but unassigned to a visible Slot, they are considered unassigned. All unassigned Promotions are considered to belong to Slot 0. To view only unassigned Promotions, click Unassigned to filter the Promotions pane to display only these associated Promotions.

Bids

Selecting Bids refreshes the pane and displays only Promotions that have a Bid for the current Event Slot Set.

The system will display the following Bid information is displayed for each Promotion to assist with planning the Event and each Slot Sets:

  • Bid Comments – Comments providing additional information to support decision making for the Bid.
  • Vendor Supported – Indicates if the Bid is supported by the vendor.
  • Bid Status – The Bid Status of ‘Submitted’, ‘Resubmitted’, ‘Cancelled’, ‘Accepted’, ‘Rejected’, or ‘Revision Requested’ will be indicated.
  • Financial Forecast – The projected financial sales for the Bid.
  • Unit Forecast – The projected unit sales for the Bid.

Selecting Edit Bid will open the Bid Details in a window.

It is possible to change the Bid Status by selecting one of the following options in the Bid Action dropdown list:

  • Accepted – Indicates that the Bid has been accepted. The Promotion should be placed on the Slot Set if this action is taken.
  • Rejected – Indicates that the Bid has been rejected. If desired, enter Comments as well explaining why the Bid was rejected.
  • Revision Requested – Indicates that the Bid requires further information before a decision can be made. Include an explanation of the revisions requested in the Comments field.

Enter Comments to provide any further information on the Bid Action taken.

After making any necessary changes, click Ok to save the updated Bid.

Version Prompt

The user can toggle Version Prompt. When activated, the user is prompted to select which Geography to associate with the Promotion, every time the user clicks Add to Event. For more information, see Slot Versioning by Geography.
