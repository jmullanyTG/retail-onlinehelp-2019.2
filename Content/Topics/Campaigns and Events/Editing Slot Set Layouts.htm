Editing Slot Set Layouts

Layout Edit Mode, available from the Layout view, is the only place where you can edit the design of event slot slots. From here, you can customize slots, their positions, space assignment, and item display format assignment. Certain actions may be restricted based on user rights.

To toggle entering/exiting Layout Edit Mode from the Layout view:

  • Click the Layout Edit button (located about the slot set preview, next to the Zoom slider).

Once in Layout Edit mode, the toolbar actions change and new panes appear in the right side of the Layout view.

  • The Slot Information pane, located in the top-right of the view, shows property details of the selected slot.

  • The Space Allocation pane, located in the bottom-right of the view, shows space allocation by slot set and the overall event.

Private Promotions that are not shared with the logged in user

Promotions that are private to the logged in user cannot be split or merged. Promotions that are adjacent to a private Promotion cannot be merged to the private Promotion. For example, a public Promotion to the right of a private Promotion cannot be merged left and the purple line does not appear. Also, logged in users cannot split a Promotion that is private to them.

Promotions in the Slot Set that have not been shared with the logged in user are indicated by the following:

Actions

You can perform the following actions while in Layout Edit Mode.

Action

Description
Edit slot properties In the Slot Information pane, you can edit the following properties of a selected slot:
  • Slot name
  • Event space

  • Item display format

    Clicking on this field displays a dropdown list populated with the available Item Display formats. The available Item Display Formats are filtered to be within corporate compliance to ensure consistent messaging to customers via the Slot Set Template Configuration screen.

    NOTE: If an Item Display Format is assigned that is no longer applicable to the Slot Set Template the option will remain until it has been substituted with a different Item Display Format. The invalid Item Display Format will be no longer available once another Item Display Format has been selected and saved.

  • Notes

NOTE: The above properties cannot be modified by the logged in user if the slot contains a Promotion that has not been shared with the logged in user. In this case, Slot Name and Notes are masked and Event Space and Item Display Format are read-only.
Display headers Clicking the button next to the Zoom slider toggles showing/hiding slot hearers, which display the position and item display format for each slot.
Display dimensions Clicking the button next to the Zoom slider toggles showing/hiding each slot's dimensions and relative size.
Drag promotions to different slots

Dragging a promotion from one slot to another is possible within the same slot set, as long as the promotion has at least one site cluster that matches the destination slot's geographies. Dragging removes the promotion from the original slot.

NOTE: Drag and drop functionality for moving around slots is disabled if either the source or target slot has a private promotion assigned.
Renumber slots

Clicking the # button next to the Zoom slider toggles entering and exiting slot renumbering mode.

NOTE: This functionality is disabled if the slot contains a Promotion that has not been shared with the logged in user.

Save changes The Save toolbar button saves changes made to the slot set.
Split a slot

Splits the selected slot into multiple slots. Slots cannot be split into widths or heights smaller than 48 DIUs (about 1/2 inch).

NOTE: This functionality is disabled if the slot contains a Promotion that has not been shared with the logged in user.

Merge slots Merges selected slots.
Equal size

The Equal Size toolbar button resizes selected slots into equal heights or widths across their combined area. Slots must be adjacent to each other and aligned vertically or horizontally.

NOTE: This functionality is disabled if at least one Promotion is selected on the Slot Set that has not been shared with the logged in user.
Clear all

The Clear All toolbar button removes all slot configurations and resets the slot set to a single slot with default settings.

NOTE: This functionality is disabled if the Slot Set contains at least one Promotion that has not been shared with the logged in user.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the slot set.
